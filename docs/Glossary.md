# Glossary

Definitions of terms used throughout the CampusSync system and documentation.

| Term | Description |
|---|---|
| **Academic Year** | A defined school session (e.g., 2025–26). Data is typically isolated per academic year. |
| **Fee Structure** | Predefined rate configuration for fee heads, usually class-specific. |
| **Ledger** | A transaction-based accounting record for students or the institution. |
| **RBAC** | Role-Based Access Control. A method of regulating access to computer or network resources based on the roles of individual users within an enterprise. |
| **School Admin** | The primary administrative authority for a single school instance. |
| **GR Number** | General Register Number. A unique permanent identification number assigned to a student at the time of admission. |
| **Installment** | A part of a fee payment made at specific intervals as defined in the [[Fee Structure Engine]]. |
| **Concession** | A reduction in fees granted to a student based on specific criteria (e.g., sibling discount, merit). |
| **Audit Log** | A chronological record of security-relevant chronological records that provide documentary evidence of the sequence of activities that have affected at any time a specific operation, procedure, or event. |
