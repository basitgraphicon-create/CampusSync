# User Classes and Roles

CampusSync utilizes Role-Based Access Control ([[Role-Based Access Control|RBAC]]) to manage permissions. Below are the primary user classes and their responsibilities.

## Role Hierarchy

1. **[[Super Admin]]**: Global system controller with access to all schools and system configurations.
2. **[[School Admin]]**: Full control over a single institution's operational and financial data.
3. **[[Principal]]**: Focuses on academic supervision, teacher performance, and result approvals.
4. **[[Accountant]]**: Manages fee collections, payroll, and financial reports.
5. **[[Teacher]]**: Responsible for attendance, marks entry, and daily academic activities.
6. **[[Staff]]**: Non-teaching personnel managing HR-linked operations.
7. **[[Parent]]**: Viewing-only access to their children's attendance, fees, and results.
