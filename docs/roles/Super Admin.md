# Super Admin Role

The **Super Admin** is the highest level of authority in the CampusSync ecosystem.

## Primary Responsibilities
- **Multi-school Management**: Creating and managing multiple school instances within the system.
- **Global Configuration**: Defining system-wide settings, default academic year structures, and global roles.
- **License Management**: Handling subscriptions and feature toggles for individual schools.
- **System Maintenance**: Database backups, system updates, and high-level audit oversight.
- **User Management**: Creating School Admins and resetting administrative credentials.

## Access Level
- **Full Read/Write Access** to all modules across all schools.
- Ability to override local school settings if necessary.
