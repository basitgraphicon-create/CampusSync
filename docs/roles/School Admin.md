# School Admin Role

The **School Admin** is responsible for the day-to-day configuration and oversight of a single educational institution.

## Primary Responsibilities
- **Academic Setup**: Defining classes, sections, and subjects via [[Academic Configuration]].
- **Fee Management**: Setting up the [[Fee Structure Engine]] for the academic year.
- **Staff Oversight**: Managing staff records and payroll approvals in [[Staff Management]].
- **Policy Enforcement**: Configuring passing rules, attendance thresholds, and communication policies.
- **Reporting**: Access to all financial and academic reports for the school.

## Access Level
- **Full Control** over all modules within their assigned school.
- Cannot access data of other schools in a multi-tenant environment.
