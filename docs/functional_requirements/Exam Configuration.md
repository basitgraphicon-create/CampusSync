# Exam Configuration

## Overview
The Exam Configuration module allows for the flexible definition of the school's examination structure and grading policies.

## Key Features
- **Multiple Exam Types**: Support for Unit Tests, Half-Yearly, Annual Exams, etc.
- **Term Grouping**: Ability to group exams into Academic Terms (e.g., Term 1, Term 2).
- **Subject Mapping**: Linking exams to specific subjects and classes via [[Academic Structure Schema]].
- **Passing Rules Configuration**: Define minimum marks or grades required to pass a subject or term.

## Implementation Details (To be expanded)
- Flexible grading scales (e.g., A1, A2, B1...).
- Rules for calculating 'Weighted Average' across different exam types.
- UI for defining exam schedules and time slots.
