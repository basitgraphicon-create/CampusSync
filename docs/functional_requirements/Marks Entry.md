# Marks Entry

## Overview
This module provides a secure and efficient way for teachers to enter and manage student performance data.

## Key Features
- **Subject-wise Entry**: Interface designed for entering marks/grades for all students in a class for a specific subject.
- **Support for Grade or Marks**: Toggle between numerical marks and alphabetical grades based on configuration.
- **Submission Locking**: Once marks are finalized and submitted, they are locked to prevent accidental or unauthorized changes.
- **Audit Logs for Modifications**: Every change made to marks (especially after locking) is logged with the user ID and timestamp.

## Implementation Details (To be expanded)
- Bulk upload capability (e.g., from CSV/Excel).
- Real-time validation for marks (e.g., entered marks cannot exceed max marks).
- Role-based permissions to allow only authorized teachers/admins to enter or modify marks.
