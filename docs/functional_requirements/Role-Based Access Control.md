# Role-Based Access Control

## Overview
The RBAC module ensures the security and integrity of the system by restricting access to features and data based on the user's role.

## Key Features
- **Define Roles and Permissions**: Create roles such as Super Admin, School Admin, Teacher, Accountant, etc., with specific access levels.
- **Restrict Module Access**: Enable/disable entire modules for specific roles based on their occupational needs.
- **Restrict Action-Level Permissions**: Finer control over actions within a module (e.g., a teacher can View results but not Delete them).
- **Maintain Audit Logs**: Comprehensive logging of sensitive actions (e.g., data deletion, permission changes) showing Who did What and When.
