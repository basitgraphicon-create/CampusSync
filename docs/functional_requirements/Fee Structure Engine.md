# Fee Structure Engine

## Overview
The Fee Structure Engine is the core of the financial module, allowing schools to define complex and flexible billing configurations.

## Key Features
- **Fee Group Creation**: Group fees logically (e.g., Monthly Tuition Fees, Annual Fees, Exam Fees).
- **Fee Head Creation**: Define individual fee items (e.g., Tuition, Lab, Sports, Admission).
- **Installment Definition**: Configure how fees can be paid over time (e.g., Monthly, Quarterly, Bi-Annually).
- **Late Fine Rule Configuration**: Set up automated rules for calculating fines on overdue payments.
- **Concession Handling**: Define rules for fee waivers or discounts (e.g., Staff ward, Sibling, Merit-based).
- **Academic-Year-Specific Fee Versions**: Fees can change from year to year without affecting past records.
