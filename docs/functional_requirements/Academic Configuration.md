# Academic Configuration

## Overview
This module handles the foundational setup of the school's academic hierarchy and resource allocation for each academic year.

## Configuration Items
- **Academic Year Creation**: Define the start and end dates for the school session (e.g., April 1st to March 31st).
- **Class and Section Creation**: Define the grades (e.g., Nursery to Class XII) and their respective sections (e.g., A, B, C).
- **Stream Definition**: Configure academic streams for higher secondary levels (e.g., Science, Commerce, Humanities).
- **Subject Creation**: Maintain a master list of subjects offered by the school.
- **Class-Subject Mapping**: Define which subjects are taught in which classes.
- **Teacher-Subject Allocation**: Assign specific teachers to the subjects they are responsible for teaching.
- **Class Teacher Assignment**: Appoint a primary teacher responsible for each class/section.
