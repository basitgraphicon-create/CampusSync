# Student Fee Application

## Overview
The Fee Application module automates the process of assigning fees to students based on their enrollment details.

## Key Features
- **Map Fee Structure During Admission**: Integrated step in the admission process to assign correct fee groups based on class and student category.
- **Generate Student Ledger**: Automated creation of a ledger record for each student upon enrollment, showing all due fees.
- **Support Partial Payments**: Flexibility to accept payments that are less than the total due amount.
- **Calculate Outstanding Balances**: Real-time updates to the ledger to show current due, paid, and balance amounts.
- **Apply Fine Automatically**: Automated calculation and addition of late fees to the student's outstanding balance based on predefined rules.

## Implementation Details (To be expanded)
- Support for multiple payment modes (Cash, Cheque, Online, Bank Transfer).
- Integration with thermal printers for quick receipt printing.
- Automated reconciliation reports for bank deposits.
